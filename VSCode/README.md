# Cheat Sheet

<br>

## Format code:
`Mac` Alt+Shift+F

`Windows` Alt+Shift+F

## Select language mode:
`Mac` Cmd+K + M

`Windows` Ctrl+K + M

## Open folder:
`Mac` Cmd+K+O

`Windows` Ctrl+K+O

## Run active file or selected code block in integrated terminal:
`Mac` Shift+Enter

`Windows` Shift+Enter

## Remove current line (without adding it to clipboard):
`Mac` Cmd+Shift+K

`Windows` Ctrl+Shift+K

## Comment out current line or selected code block:
`Mac` Cmd+Shift+7

`Windows` Ctrl+Shift+7

## Markdown preview:
`Mac` Cmd+Shift+V

`Windows` Ctrl+Shift+V

## View Problems: 
`Mac` Cmd+Shift+M

`Windows` Ctrl+Shift+M

## Delete matching brackets:
`Mac` Cmd+Shift+Alt+I

`Windows` Ctrl+Shift+Alt+I
