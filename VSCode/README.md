# Shortcuts:

## Format code:
`Mac` Alt+Shift+F

`Windows` Alt+Shift+F

## Markdown preview:
`Mac` Cmd+Shift+V

`Windows` Ctrl+Shift+V

## View Problems:
`Mac` Cmd+Shift+M

`Windows` Ctrl+Shift+M

## Delete matching brackets:
`Mac` Cmd+Shift+Alt+I

`Windows` Ctrl+Shift+Alt+I
