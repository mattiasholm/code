# Cheat Sheet

<br>

## Python shebang:
```bash
#!/usr/bin/env python3
```

## Check installed Python version:
```bash
python3 --version
```

<br><br>

## Print to standard output:
```python
print("Message")
```

## Add a single-line comment:
```python
#Comment
```

## Add a multi-line comment:
```python
"""
Multi-line
Comment
"""
```

## Assign a variable:
```python
x = 1
```

## Assign multiple variables at once:
```python
x, y, z = 1, 2, 3
```

## Assign the same value to multiple variables:
```python
x = y = z = 1
```
