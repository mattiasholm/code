# Cheat Sheet

<br>

## Python shebang:
`#!/usr/bin/env python3`

```bash
#!/usr/bin/env python3
```

## Check installed Python version:
`python3 --version`

```bash
python3 --version
```

<br><br>

## Print to standard output:
`print("Message")`

## Add a single-line comment:
`#Comment`

## Add a multi-line comment:
```python
"""
Multi-line
Comment
"""
```

## Assign a variable:
`x = 1`

## Assign multiple variables at once:
`x, y, z = 1, 2, 3`

## Assign the same value to multiple variables:
`x = y = z = 1`
